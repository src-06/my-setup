# My Arch Setup
Just my setup installation for Arch Linux. Setup with Nix (Flake + Home Manager) + Noctalia Shell + Niri WM + My Nix and Niri configurations. 

## Installations
Clone this repo and install with setup

```sh
git clone https://github.com/src-06/my-arch-setup.git
cd my-arch-setup
./setup install
```

## Configuration
First open Qt6 Settings.

- On Appereance:
  - Style -> Fusion
  - Color scheme -> noctalia
  - Standard dialog -> GTK3
- On Fonts:
  - General -> CodeNewRoman Nerd Font 11
  - Fixed width -> JetBrainsMono Nerd Font Mono 11
- On Icon Theme
  - Select Papirus-Dark
- Apply and OK

Then open GTK Settings.

- On Widgets
  - Default font -> CodeNewRoman Nerd Font 11
  - Color scheme -> Prefer dark
- Icon theme
  - Select Papirus-Dark
- Apply

And on other option you can custom as you wish.
